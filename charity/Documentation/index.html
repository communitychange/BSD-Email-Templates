Using Mailchimp Templates Directly

STEP 1: Preparing the templates before importing:

  • Open template folder -> Mailchimp folder -> mailchimp.html -> Open with notepad++.
  • Use above STEP 2,3,4,5
  • Select images folder and mailchimp.html u wish to import and zip up together and give zip a name without spaces.

    • Note 1

    You can also use above .html downloaded using online template builder,for uploading to mailchimp

    Note 2

    You can replace images after importing template in mailchimp template builder OR as explained in previous step 4

STEP 2: Importing & Saving the template:

  • Staring in your dashboard, click on Templates.
  • Press Import Zip File.

  • When dialog box opens select browse to import the previously zipped files and press upload.

  • Save,from bottom menu

STEP 3: Using & Editing template Styles

  • Create Campaign
  • Select a Template->Saved Templates->Select
  • Edit styles

  • Proceed further steps for sending
